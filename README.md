ChatApp
=======

Javaの練習ために作ったターン制コンソールチャットアプリケーション

サーバー側として起動する
---------------------------
ポート番号10000で待ち受け開始
```
$ java bin/console.ConsoleApp 10000
```

クライアント側として起動する
------------------------------
127.0.0.1:10000に接続する
```
$ java bin/console.ConsoleApp 127.0.0.1 10000
```

終了方法
---------
どちらかが「007」と発言すると接続が切断されアプリケーションが終了する
